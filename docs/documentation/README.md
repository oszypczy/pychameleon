# Projekt implementacji algorytmu CHAMELEON

**Autor:** Oliwier Szypczyn
**Przedmiot:** Metody Eksploracji Danych, sem. letni 2025/2026, Politechnika Warszawska
**Prowadzący:** dr inż. Robert Bembenik
**Algorytm:** CHAMELEON — Karypis G., Han E., Kumar V., *IEEE Computer* 32(8), 1999, str. 68–75

Niniejszy dokument stanowi **projekt implementacji** algorytmu CHAMELEON na Etap 1
projektu MED (deadline 08.05.2026). Zgodnie z wymaganiami prowadzącego
(`Projekt MED_2026l_rb.pdf`, str. 1–2) zawiera:

- **strukturę implementacji** z odniesieniem do kroków algorytmu (rozdz. 3),
- **struktury danych** dla danych źródłowych i algorytmu (rozdz. 4),
- **ocenę efektywności** proponowanego rozwiązania (rozdz. 5).

Kompilacja PDF: `bash scripts/build_docs.sh`. Kod źródłowy: `src/pychameleon/`.
